        \dot{\mf{x}}( n \Delta ) &= \mf{A} \mf{x}(n \Delta) + \mf{B}\mf{u}(n \Delta) \\

    Starting from the value of the start at time $0$, $\mf{x}\lp 0 \rp$, we can numerically compute the evolution of the state for a given input $\mf{u}\lp t \rp$.
